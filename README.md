# r4tur1 — Terminal Portfolio

> An interactive, terminal-styled personal portfolio built with HTML, CSS, and Vanilla JavaScript.

**Live site → [r4tur1.github.io](https://r4tur1.github.io)**

---

## 🖥️ Overview

This is an upgraded, interactive CLI-style portfolio designed to showcase my journey as a developer. It draws inspiration from retro terminals while maintaining a modern, clean aesthetic.

### ✨ Features
- **Interactive Terminal:** Fully functional command-line interface.
- **Boot Sequence:** Simulated system initialization with ASCII art.
- **Command History:** Use ↑ and ↓ keys to navigate previous commands.
- **Tab Completion:** Quickly finish commands with the Tab key.
- **Responsive Design:** Works on mobile (though best experienced on desktop).
- **CRT Effect:** Subtle scanlines and glow for that authentic terminal feel.

---

## 🛠️ Commands

| Command | Description |
|---------|-------------|
| `help` | List all available commands |
| `whoami` | Learn about my background and goals |
| `skills` | View my technical stack with proficiency bars |
| `projects` | Interactive list of my work |
| `education` | My academic history |
| `contact` | Get my social links and email |
| `clear` | Clean the terminal screen |
| `gui` | Switch back to the classic portfolio view |
